Além disso, agora podemos usar ambos "apt" e "dpkg" para instalar o pacote. Como já trabalhamos com "apt", vamos nos voltar ao "dpkg" para o próximo exemplo.

```
username@hostname[~]$ sudo dpkg -i strace_4.21-1ubuntu1_amd64.deb

```
